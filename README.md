# MyDictionary

Foreign language dictionary builder


A personal project where users can:
- Create an account
- Create dictionaries (one language per dictionary)
- Paste foreign text to generate word lists
- Manually add translations per word

## Tech Stack
- Frontend: HTML, CSS, JS (GitHub Pages)
- Backend: Python (Flask)
- Database: SQLite

This project uses:

- GitHub Pages for the frontend (static HTML/CSS/JS)

- A Python backend deployed separately
